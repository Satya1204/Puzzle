v1.0.8
Update: Migrated from Old Input System to New Input System

1. Input System Installation Steps:
   - Open Unity → Window → Package Manager
   - Search for "Input System"
   - Click "Install" (recommended version 1.14.2 or newer)
   - Go to Edit → Project Settings → Player → Other Settings
   - Set "Active Input Handling" to "Input System Package (New)"
   - Restart Unity after applying changes.

2. Scene Changes:
   - Scene Path: WaterSortPuzzleColorGame/Scenes/Menu
   - In the "EventSystem" GameObject:
     - Remove: Standalone Input Module
     - Add: Input System UI Input Module

3. Script Updates:
   - Updated scripts to use the new Input System API:
     - WaterSortPuzzleColorGame/Scripts/GameManager.cs
     - WaterSortPuzzleColorGame/Scripts/GameController/ActionController.cs
     - WaterSortPuzzleColorGame/Scripts/Tutorial/ClickController.cs
  - Add New Folder: HexaSortPuzzleGame/Scripts/Input

v1.0.3

This update focuses on improving responsive UI support, better code structure, and maintainability. Please follow the note at the end for a fresh setup.

1. Scripts

- CanvasController.cs
  - Added UpdateCanvasMatch() function.
  - Modified CanvasScaler settings to support responsive UI.

- LevelMakerCreateBottle.cs
  - Added RepositionBottles() to calculate bottle positions in landscape mode.

- GameManager.cs
  - Updated the Start() method for improved initialization.

- MapPanel.cs
  - Updated to support smooth scrolling and responsive UI adjustments.

- ScreenResizeManager.cs
  - New script to handle responsive UI resizing across different screens.

- Button Click Listeners
  - Moved all button listeners from Inspector to scripts for better prefab consistency and reliability.

- Obsolete Script Cleanup
  - Removed:
    - ShowHidable.cs
    - IShowHideable.cs
    - DestroyToasterMessage.cs
    - NavigationBar.cs

- BackHandler.cs
  - Created to manage all back navigation and event handling centrally.

- New Folder Structure
  - Scripts/Navigations/ → Navigation-related scripts.
  - Scripts/Popups/ → All popup-related scripts now organized separately.

- PopupUtils.cs
  - Updated to reflect the new popup system.

2. Prefabs

- FreeCoinPopup
  - Removed DestroyToasterMessage component from the Inspector.

3. UI Hierarchy Changes

- TopList
  - Adjusted CoinButton position for responsive layout.

- HomePanel
  - Updated all element positions and sizes for better responsiveness.

- BottomList
  - Redesigned to support responsive UI.

- CollectionPanel / ShopPanel / MapPanel
  - Updated popup and child object positions for responsive layout.

- ScreenResizeManager
  - Added as a new GameObject in the hierarchy.
